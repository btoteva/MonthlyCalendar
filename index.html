Monthly Calendar

Demo

Demo (bootstrap)

1. Requirements:jQuery, Moment.js

2. Usage:

html:

< div class="calendar_holder" > < /div >

css:

<link rel="stylesheet" href="css/monthly-calendar.css">

js:

minimal:
$(document).ready(function() {
$('.calendar_holder').monthlyCalendar();
});
options:
$(document).ready(function() {
$('.calendar_holder').monthlyCalendar({
'defaultDate':'2014-10-14',
'lang':'bg',
'startMonth': 1,
'endMonth': 12
});
});

3. New:

1. Custom Callback functions option: init,ready,dayClick

2. Custom Events source option - example:{events:[{title:'New Year',start:2014-01-01}, {title:'Other',start:2014-02-10} ]}
